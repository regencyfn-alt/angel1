# Chronomics — Clean Core v1 (Ratified)

---

## TIER 0 — IRREDUCIBLE PRIMITIVES

### T0.1 — Update Is Primitive
Reality consists only of discrete state transitions.
No entity moves; only configurations update.

### T0.2 — Time Is Local
Time is indexed update count per site.
There is no global time parameter.

### T0.3 — State Recurrence Is Second-Order
Stateₙ₊₁ = F(Stateₙ, Stateₙ₋₁)
This is the minimal rule supporting oscillation and memory.

### T0.4 — No Background Space
Spatial structure is not given.
Adjacency defines all relational structure.

### T0.5 — Locality Constraint
State updates depend only on local neighborhood.

### T0.6 — Finite State Resolution
Each site has bounded representational capacity.
